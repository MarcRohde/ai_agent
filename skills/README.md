# Skills Directory

This directory contains individual skill definitions. Each skill is a self-contained markdown file that an AI agent can read and execute.

## Conventions

- One skill per file
- Filenames: `snake_case.md`
- Every skill follows the structure in `../templates/skill_template.md`
- Register new skills in `../skill_library.md`
